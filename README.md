# playwright-banking-framework
Designed and implemented a scalable end-to-end UI automation framework using Playwright and JavaScript following the Page Object Model (POM) architecture to improve maintainability, modularity, and reusability. Automated key banking workflows including customer onboarding, account creation, deposits, withdrawals, transaction validation, and balance verification. Implemented reusable page classes, dynamic test data generation, robust locator strategies, and assertion-based validations to ensure reliable test execution.
